Как получить Telegram ID?

Для получения Telegram ID пользователя:

  1. Попросите пользователя написать боту @userinfobot
  2. Бот ответит информацией, включая числовой ID
  3. Скопируйте этот ID в поле выше

Также ID можно получить программно через Telegram Bot API или веб-версию Telegram.
