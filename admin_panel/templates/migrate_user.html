Как получить Telegram данные?
Через @userinfobot:
  1. Попросите пользователя написать @userinfobot
  2. Бот пришлет ID и другую информацию
  3. Скопируйте данные в форму выше
Через веб-версию Telegram:
  1. Откройте web.telegram.org
  2. Найдите пользователя
  3. ID будет в URL: https://web.telegram.org/k/#ID
Важно: Убедитесь, что Telegram ID принадлежит именно этому пользователю!
